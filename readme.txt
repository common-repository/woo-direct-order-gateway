== Description ==

Plugin will change the checkout page and hide all current enabled payment gateway. It will enable to checkout without payment.

Plugin will help to add additional fee before sending the item. The admin will send an updated order invoice to customer via email for payment before processing the item.

Plugin Features: 

1. Hide enabled payment gateway
2. Direct order without payment
3. Admin can add additional fee
4. Admin will send an updated invoice to customer
5. Payment is enabled after sending an updated invoice to customer
6. Tested up to latest Wordpress

== Installation ==

This section describes how to install the plugin and get it working.
1. Download and install the plugin from WordPress dashboard. You can also upload the entire "Woo Direct Order Gateway" folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done
